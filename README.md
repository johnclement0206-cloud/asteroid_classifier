# asteroid_classifier
A simple Python program using PyTorch to train and make models based off the NeoWs (Near Earth Object Web Service) API offered by NASA.

- uses DEMO_KEY (limited to 50 API calls per day, or 30 per hour) as the API key, but you can request your own API key at https://api.nasa.gov that gets a standard hourly rate of 1000 calls.
